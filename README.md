# PawMart – Pet MarketPlace

**PawMart** is a modern, responsive web application that allows users to buy, sell, and explore pet items, including pet food, accessories, toys, and more—all in one convenient platform.

Live URL: https://pawmart-knox.surge.sh  


## Key Features

- Pet Listings: Browse the latest pet products and accessories with clean, detailed listing pages. 
- Fully responsive design – looks perfect on mobile, tablet, and desktop  
- Google Sign-In authentication (powered by Firebase)   
- Buy & Place Orders: Add items to your cart and place orders seamlessly. 
- Protected Routes: Smooth page load and secure access for logged-in users.    
- Beautiful UI with better UX 

## Tech Stack & npm Packages Used

 1. `react` + `react-dom` Core framework                           
 2. `react-router-dom` Client-side routing & protected routes    
 3. `tailwindcss` + `daisyui` + `react-toastify` + `motion` + `swiper` Design and styling                     
 4. `lucide-react` + `react-icons` Beautiful icons  
 5. `firebase` Google authentication & Firestore         
 6. `surge` deployment  
 7. `express` custom apis
 8. `mongodb` data management
